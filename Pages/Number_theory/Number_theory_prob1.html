Hossam and Combinatorics

Link : https://codeforces.com/problemset/problem/1771/A

Problem summary:

We have an array of n positive integers. We have to choose pairs (i,j) such that
        The absolute difference |ai−aj| must be equal to the maximum absolute difference over
        all the pairs in the array. More formally, |ai−aj|=max 1≤p,q≤n |ap - aq|.

Solution

We will have to just count cnt1 = number of elements equal to the minimum element(mn)
          and cnt2 = number of elements equal to the maximum element(mx)
          now, number of ways to choose n pairs = cnt1*cnt2 * 2 (orderder pairs (i,j)is not same as (j,i))
          but if mx = mn the answer = (no. of pairs)*2 (ordered pairs)
          no_of_pairs = cnt1C2 = cnt1*(cnt1-1)/2
